## Why another http upload server?
No one is usable on npm.

## Installation

```sh
npm install -g simple-http-upload-server
```

## Basic usage

```sh
# change into the directory for uploading files
shell> mkdir /tmp/upload
shell> cd /tmp/upload

# launch server
shell> simple-http-upload-server
```
